Styles

PushFeedback allows users to seamlessly integrate feedback mechanisms into their web pages while providing the flexibility to style and customize its appearance. This reference document lists all the available CSS custom properties that can be overridden to match your website's design.

Usageâ€‹

To customize these properties, define them in your website's CSS using the :root selector for global changes or a specific selector to target particular areas:

:root {
    --feedback-primary-color: #00D68F;
    ...
}

Propertiesâ€‹

The tables below lists all the available properties you can override:

Global propertiesâ€‹

PropertyDescription
--feedback-primary-colorDefault value: #0070F4
Primary color used for important elements and feedback.
--feedback-secondary-colorDefault value: #fbfbfb
Secondary color used for UI backgrounds and secondary elements.
--feedback-light-colorDefault value: #ccc
Light color used for subtle UI elements, borders, and backgrounds.
--feedback-dark-colorDefault value: #191919
Dark color primarily used for text and dark UI elements.
--feedback-text-colorDefault value: #5f6368
General text color for feedback.
--feedback-white-colorDefault value: #fff
Standard white color used across the UI.
--feedback-font-familyDefault value: "Inter", "Tahoma", "sans-serif"
The primary font family for the feedback system.
--feedback-heading-font-sizeDefault value: 16px
Font size for headings within feedback.
--feedback-text-font-sizeDefault value: 14px
Font size for general text within feedback.
