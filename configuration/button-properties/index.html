Feedback button properties

The PushFeedback button is highly customizable, allowing you to tailor its appearance and behavior to suit your application's needs. This page lists all available properties you can use to adjust the behavior and appearance of the PushFeedback button in your application.

To customize the properties, you simply need to set the appropriate attributes on the <feedback-button> element in your HTML. For example:

```html 

The table below lists all the available properties you can set, their corresponding HTML attributes, their type, potential values (if applicable), and a brief description.

| Property (attribute)                       | Type     | Available values                  | Description                                                           |
| ------------------------------------------ | -------- | --------------------------------- | --------------------------------------------------------------------- |
| `buttonPosition` (`button-position`)       | `string` | `'default'`, ...                  | Position of the button. Default is `'default'`.                       |
| `buttonStyle` (`button-style`)             | `string` | `'default'`, ...                  | Style of the button. Default is `'default'`.                          |
| `email` (`email`)                          | `string` |                                 | The default email address. Empty by default.                          |
| `emailPlaceholder` (`email-placeholder`)   | `string` |                                 | Placeholder for the email input.                                      |
| `errorModalTitle` (`error-modal-title`)    | `string` |                                 | Title for the error modal.                                            |
| `hideIcon` (`hide-icon`)                   | `boolean`| `true`, `false`                  | Whether to hide the button icon.                                      |
| `hideScreenshotButton` (`hide-screenshot-button`) | `boolean` | `true`, `false` | Whether to hide the screenshot button.                                |
| `messagePlaceholder` (`message-placeholder`) | `string` |                             | Placeholder for the feedback message input.                           |
| `modalPosition` (`modal-position`)         | `string` | `'center'`, ...                  | Position of the feedback modal.                                       |
| `modalTitle` (`modal-title`)               | `string` |                                 | Title of the feedback modal.                                          |
| `project` (`project`)                      | `string` |                                 | The project ID. Required.                                            |
| `screenshotButtonTooltipText` (`screenshot-button-tooltip-text`) | `string` |     | Tooltip text for the screenshot button.                               |
| `screenshotTopbarText` (`screenshot-topbar-text`) | `string` |                       | Text shown on the screenshot's top bar.                               |
| `sendButtonText` (`send-button-text`)      | `string` |                                 | Text for the send feedback button.                                    |
| `successModalTitle` (`success-modal-title`) | `string` |                                | Title for the success modal.                                          |

For properties with `...` under "Available values", please consult the main PushFeedback documentation or the specific property's documentation to see the full list of available options.
