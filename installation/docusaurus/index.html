Docusaurus

This guide walks you through the process of adding PushFeedback to your Docusaurus website.

Prerequisites​

  • A Docusaurus site set up and running.
  • Familiarity with Docusaurus configuration and structure.
  • Your PushFeedback project ID.

1. Install PushFeedback via npm​

Navigate to the root directory of your Docusaurus project using your terminal or command prompt. Install pushfeedback with:


2. Add PushFeedback Styles and Script​

In Docusaurus, the best place to add global styles and scripts is in the docusaurus.config.js file. Add the following:

a. Add to the head Array​

Find or add the head array in docusaurus.config.js and append the following:

  tagName: 'link',
  rel: 'stylesheet',
  href: 'https://cdn.jsdelivr.net/npm/pushfeedback/dist/pushfeedback/pushfeedback.css',
},

b. Add to the scripts Array​

Find or add the scripts array in docusaurus.config.js and append the following:

  tagName: 'script',
  type: 'module',
  src: 'https://cdn.jsdelivr.net/npm/pushfeedback/dist/pushfeedback/pushfeedback.esm.js',
},

3. Add the Feedback Button Component​

4. Build and Verify​

Rebuild your Docusaurus site and start the development server:

npm run serve

Ensure the PushFeedback button displays and functions correctly on your Docusaurus site.

Final Thoughts​

Collecting feedback on documentation and content is invaluable. With PushFeedback embedded into your Docusaurus site, you can effortlessly gather user insights and improve your content quality over time.
